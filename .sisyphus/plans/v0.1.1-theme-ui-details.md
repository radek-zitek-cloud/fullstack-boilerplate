# Theme System UI Details (v0.1.1)

## Approach: Hard-Coded Variants + Settings Page

### Why This Approach?

1. **Clean Header** - User dropdown stays uncluttered
2. **Dedicated Space** - Settings page provides room for detailed theme options
3. **Extensible** - Can add user-created themes in v0.2.0 without breaking changes
4. **Fast Implementation** - Fits v0.1.1 timeline

### User Flow

```
1. User clicks avatar/name in header
2. Selects "Settings" from dropdown
3. Lands on Settings page with tabs/sections
4. Clicks "Appearance" tab
5. Sees:
   - Theme Mode: [Light] [Dark] [System]
   - Accent Color: [Slate] [Blue] [Emerald]
   - Live preview of selected combination
6. Changes auto-save to localStorage
```

### Technical Details

**Storage**: localStorage only (for v0.1.1)
```typescript
localStorage.setItem('theme-mode', 'dark');
localStorage.setItem('theme-color', 'emerald');
```

**CSS Classes Applied**:
```html
<html class="dark theme-emerald">
```

**Color Variants Defined**:
- `theme-slate` (default gray-ish)
- `theme-blue` (professional blue)
- `theme-emerald` (fresh green)

Each variant defines:
- `--color-primary`
- `--color-primary-foreground`
- `--color-ring`
- etc.

### Future Extension (v0.2.0)

When upgrading to user-created themes:

1. Add `user_preferences` table:
   ```sql
   CREATE TABLE user_preferences (
     user_id INTEGER PRIMARY KEY,
     theme_mode VARCHAR(10),
     theme_color VARCHAR(20),
     custom_theme JSON  -- For user-created themes
   );
   ```

2. Add "Custom" option to color selector
3. Add color picker UI for custom themes
4. Migrate localStorage to database on login

### Benefits of Settings Page vs Dropdown

| Dropdown Toggle | Settings Page |
|----------------|---------------|
| Quick access | More space for options |
| Clutters header | Clean separation |
| Limited options | Room for expansion |
| Good for simple on/off | Better for complex config |

**We chose Settings Page** because theme configuration is not something users change frequently, and it deserves a dedicated space with preview and explanation.
